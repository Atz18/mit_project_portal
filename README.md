# MIT Project Portal

## Get Started

### Backend code

1. Install all dependencies

```bash
# in root directory
cd mit_project_portal
npm install
```

2. Start the development server

```bash
# in root directory
npm run start
```

### Frontend code

1. go to frontend folder

```bash
# in root directory
cd frontend
```

2. install frontend dependencies

```bash
# in frontend directory
npm install
```

2. Start the development server

```bash
# in frontend directory
npm run start
```
